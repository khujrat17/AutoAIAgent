# AutoAIAgent

**AutoAIAgent** is an AI-powered automation tool for generating and posting FullStack .NET developer content on LinkedIn. It generates trending topics, concise C# code snippets, engaging post text, and corresponding code snippet images automatically.

---

## 🚀 Features

- **Topic Generation**: Generates short, viral, and trending .NET development topics.  
- **Code Snippet Generation**: Produces concise and complete C# code snippets.  
- **LinkedIn Post Text**: Generates professional, high-engagement post text including explanations and curiosity hooks.  
- **Code to Image Conversion**: Converts code snippets into visually appealing images using multiple online tools:
  - Carbon
  - Codeimg.io
  - Code Snapshot
  - Writecream Code-to-Image  
- **Hashtag Automation**: Generates relevant hashtags to increase post reach.  
- **LinkedIn Posting**: Automates posting with caption + image directly to LinkedIn.  
- **Retry & Logging**: Automatically retries failed steps and logs detailed information.

## 💡 How it Works

1. **Generate Topic** – AI generates a short, trending .NET topic.  
2. **Generate Code** – AI produces a concise C# snippet for the topic.  
3. **Generate Post Text** – AI generates LinkedIn-ready post text with explanation & emojis.  
4. **Generate Image** – Converts code snippet into an image via online tools.  
5. **Generate Hashtags** – AI suggests relevant hashtags.  
6. **Post to LinkedIn** – Uses Playwright to post the text + image automatically.

## ⚙️ Setup

1. Clone the repository:

```bash
git clone https://github.com/khujrat17/AutoAIAgent.git
cd AutoAIAgent


Install dependencies via NuGet:

dotnet restore


Configure LinkedIn credentials and OpenRouter API key in appsettings.json:

{
  "LinkedIn": {
    "Email": "YOUR_EMAIL",
    "Password": "YOUR_PASSWORD"
  },
  "OpenRouter": {
    "ApiKey": "YOUR_OPENROUTER_API_KEY"
  }
}


Run the project:

dotnet run


The service will generate a post and publish it to LinkedIn automatically.

⚠️ Notes

Ensure LinkedIn credentials are correct. Two-factor authentication may block automation.

Generated code snippets are concise; review them before sharing publicly.

UI changes on online tools may require updating the CodeImageService.
